# Busy Program

这是一个自我修改的实验性项目，通过 GitHub Actions 定时运行并修改自身代码。

## 功能

- 每次运行时扫描当前目录的所有文件
- 使用 DeepSeek API 分析文件内容并生成修改
- 随机添加有趣的功能和注释
- 记录每次运行的活动日志

## 运行机制

项目通过 `main.py` 文件驱动，每次运行时：

1. 收集当前目录所有文件内容
2. 发送给 AI 模型进行分析
3. 根据 AI 的建议修改文件
4. 记录运行日志

## 注意事项

- 这是一个实验性项目，代码会随时间变化
- 不要在生产环境中使用
- 仅供娱乐和学习目的
